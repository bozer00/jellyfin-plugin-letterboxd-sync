# 🧭 Design Index

This folder holds all design specifications and architectural decision records (ADRs) for the Jellybox project.

## Documents

| Document | Scope | Description |
|---|---|---|
| [[Design Decisions]] | v2 Spec | Agreed-upon decisions: matching strategy, caching, UI, sync config. |

## How to Use

When making a significant design choice that affects the project's direction, create a new note here using the format:

```
# ADR-{number}: {Title}

**Status**: Proposed / Accepted / Deprecated
**Date**: YYYY-MM-DD

## Context
Why is this decision needed?

## Decision
What was decided?

## Consequences
What are the trade-offs?
```

Link design decisions back to their relevant plugin with `[[Technical Architecture]]` wikilinks.
